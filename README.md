# Shop bán quần áo nữ
# Bài tập lớn môn: Thực tập cơ sở ngành
### Mục đích: Tạo ra website bán hàng cơ bản sử dụng Node.js và SQL Server
### Công nghệ sử dụng:
+ Front-end: HTML, CSS, JavaScript
+ Back-end: Node.js
+ Cơ sở dữ liệu: SQL Server
+ Công cụ: Visual Studio Code, SQL Server Management Studio
### Tính năng:
**- Trang chính:**
  + Đăng ký/Đăng nhập
  + Xem danh sách sản phẩm theo danh mục, bộ sưu tập
  + Xem chi tiết sản phẩm
  + Chỉnh sửa giỏ hàng
  + Đặt hàng

**- Trang quản trị:**
  + Đăng nhập
  + Thống kê doanh thu (Theo ngày/tháng, theo phương thức thanh toán)
  + Quản lý người dùng (CRUD)
  + Quản lý sản phẩm (CRUD)
  + Quản lý đơn hàng (CRUD)
  + Quản lý bộ sưu tập (CRUD)
